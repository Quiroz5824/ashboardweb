Bienvenido

En la Universidad Politécnica de Texcoco, surgió la necesidad de fortalecer los procesos de gestión académica mediante el aprovechamiento de herramientas tecnológicas actuales. Con este propósito, se planteó el desarrollo de un Dashboard web interactivo, concebido como una solución integral para recopilar, analizar y visualizar datos relacionados con la gestión administrativa.
